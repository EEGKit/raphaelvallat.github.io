Bootstrapped confidence intervals

Bootstrapping is a data-driven procedure which generates a sampling distribution by repeatedly taking random samples from the known sample with replacement. The main benefit of the bootstrap is that it allows to set confidence intervals on parameters (e.g. mean, standard deviations, effect sizes) without having to make a priori assumptions.
